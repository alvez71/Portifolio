Sobre Mim

Olá! Sou um desenvolvedor front-end em constante evolução, apaixonado por transformar linhas de código em experiências digitais funcionais e intuitivas. Atualmente, venho aprimorando meus conhecimentos em tecnologias fundamentais como HTML5, CSS3 e JavaScript (ES6+), sempre com foco em escrever um código limpo e organizado.

Um dos meus maiores desafios recentes tem sido liderar o desenvolvimento de uma plataforma de e-commerce completa para a venda de chocolates na minha escola. Neste projeto, não atuo apenas na interface; estou integrando o sistema com o Supabase para gerenciar toda a parte de infraestrutura, incluindo o banco de dados e a autenticação segura dos usuários. Esse projeto tem sido fundamental para eu entender como conectar o front-end ao back-end de forma eficiente.

Além do desenvolvimento web, também exploro a lógica de programação com Python, criando ferramentas utilitárias que reforçam minha base matemática e de resolução de problemas. Acredito que o aprendizado contínuo é a chave para o sucesso na tecnologia, e estou sempre em busca de novos frameworks e metodologias para elevar o nível das minhas entregas. Meu objetivo atual é consolidar minha carreira como um desenvolvedor Full Stack, unindo design moderno com sistemas robustos e escaláveis.
